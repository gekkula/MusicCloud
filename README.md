



#### API usage
###### Create
@params:

*token* a unique id for session

*title* title of  music file

```
locahost:3030/create/{token}/{title}

```

###### Read single
@params:

*id* id of record
```
locahost:3030/read/{id}

```
###### Read All
```
locahost:3030/readAll/

```


###### Delete single
@params:

*id* id of record
```
locahost:3030/delete/{id}

```
